# Vegastrike Sprite Editor

Written in C++ using gtkmm-3.0.

## Building with Make
The legacy Makefile is still provided:

```bash
make
```

## Building with CMake
Alternatively, you can use CMake:

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

To install the binary to your system:

```bash
cmake --install .
```
